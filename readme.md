## Usage
1. Clone this repository
```bash
git clone https://github.com/kikuKeii/BC-street-kit.git
```
2. Install packages (use `yarn` or `npm`)
```bash
npm install 
```
3. Start server
```bash
npm run start 
```
or
```bash
npm dev (you have to install nodemon on your computer)
```

## API Documentation
__Api Path__ : localhost:3000</br>
__Try__ : https://bc-street-kit.vercel.app or https://bcstreetkit.kronjo.com

| Method | Endpoint | Params | Description |
| ------ | -------- | ------ | ------------|
| GET | / | - | Index |
| GET | /api | - | index of API |
| GET | /api/users | - | Index of Users |
| GET | /api/users/s?uid={query} | ueser ID | index of Users with ID |
| GET | /api/author | - | index of author |
